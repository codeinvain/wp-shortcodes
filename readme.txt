Contributors: CodeInVain
Requires at least: 2.5.1
Tested up to: 3
Tags: wordpress, shortcode

Allows the use of a simple shortcode to embed in a post or page.

== Description ==
embeding gist with gist id using
[gist id=468329]
embeding iframe using
[ifrmae src=<url> width=<width> height=<height>]
== Installation ==


3) Upload the directory ws_shortcode to your wordpress plugin directory (/wp-content/plugins).

4) Activate the plugin.

5) Use shortcodes in your posts or pages
